Koncat App

Koncat is a powerful platform designed to streamline student data management and recruitment. It simplifies job matching for students, enables institutions to track progress, and helps employers identify top talent efficiently. With an AI-driven job-matching algorithm and a scalable Firebase backend, Koncat ensures seamless experiences across mobile and web.

Whether you're a student, educator, or recruiter, Koncat empowers you with data-driven insights and personalized recommendations for a smarter recruitment process.

Korridor - Job Posting Portal

Korridor is a recruitment platform designed to streamline the hiring process for recruiters at KIIT University. It provides a centralized system where recruiters can efficiently sort, filter, and select students for placements and internships based on various criteria such as skills, academic performance, and project experience.

The platform features a user-friendly web interface developed using Flutter, integrating with a backend database that stores detailed student profiles. Recruiters can leverage customizable checkbox filters, dynamic search functionalities, and predefined quick-access filters to refine their candidate selection. Additionally, Korridor allows recruiters to create and save customized lists, export student data in multiple formats (CSV, PDF), and submit job descriptions effortlessly.

With secure authentication via Firebase, real-time filtering, and a feedback mechanism for continuous improvements, Korridor ensures a seamless recruitment experience. It is designed to cater to different types of recruiters, from mass hiring firms to specialized tech recruiters, enhancing efficiency and precision in the hiring process.
